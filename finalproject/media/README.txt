MEDIA FILES DIRECTORY
=====================

This directory should contain the following media files:

1. restaurant-video.mp4 (or .webm)
   - A 30-second video showcasing the restaurant
   - Should be optimized for web (preferably under 5MB)
   - Formats: MP4 (H.264) or WebM

2. restaurant-audio.mp3 (or .ogg)
   - A 30-second audio clip (e.g., founder's message, restaurant ambiance)
   - Should be optimized for web
   - Formats: MP3 or OGG

NOTE: These files are referenced in:
- story.html (video)
- menu.html (audio)

Please add your actual media files to this directory with the exact filenames above.
