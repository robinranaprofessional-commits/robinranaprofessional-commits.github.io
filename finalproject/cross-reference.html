4. Minimum Requirement Features

4.1 HTML5 Features

  • HTML5 structural tags (header, nav, main, section, article, footer)
  • Lists (ul, ol, dl) - Used in navigation, menu, and story pages
  • Special characters (—, &, ©, –)
  • Meta tags (description, keywords, author, viewport, robots)
  • Footer with last updated date (JavaScript-powered)
  • Inline images (static and clickable)
  • Blockquote element

4.2 CSS3 Features

  • Contextual selectors (e1, e2, e3; e f; e > f; e + f)
  • Attribute selectors ([href*="contact"], [alt*="Logo"])
  • RGBA & HSLA colors (used throughout)
  • Dynamic pseudo-classes (:link, :visited, :hover, :active, :focus)
  • Structural pseudo-classes (:first-of-type, :last-of-type)
  • Background styles (background-size, background-clip, background-origin)
  • Drop cap on home page (::first-letter)
  • Grid layout (features-grid on index.html)
  • Flex layout (story-article on story.html)
  • Float layout (menu-item on menu.html)
  • Positioning layout (contact-info on contact.html)
  • Text and box shadows
  • Transform rotate (logo hover, feature cards)
  • Opacity (two semi-transparent images in feature cards)
  • Transitions and animations (fadeInDown, fadeInUp, pulse)

4.3 Web Form Features

  • HTML5 form elements (email, tel, date, number, textarea, select, checkbox)
  • Form styling (fieldsets, legends, focus states)
  • Inline validation (using :focus pseudo-class and JavaScript)
  • Form validation (pattern attributes, required fields)

4.4 Multimedia Features

  • Video element (30-second restaurant video)
  • Audio element (30-second audio clip)
  • CSS transitions (applied to multiple elements)
  • CSS animations (fadeInDown, fadeInUp keyframes)

4.5 JavaScript Features

  • Variables (let, const)
  • Operators (arithmetic, comparison, logical)
  • Functions (updateDateTime, validateForm, toggleMobileNav)
  • Arrays (menuItems, specialDays, contactMethods)
  • Conditional statements (if/else, ternary operators)
  • System date/clock (real-time date/time display)
  • Countdown timer (countdownToEvent function)
  • Form validation (JavaScript validation with alerts)
  • JavaScript alerts (validation error and success messages)

4.6 Responsive Design Features

  • Media queries (desktop, tablet, mobile breakpoints)
  • Contextual selectors for menus (sub-level menu styling)
  • Navicon for mobile (hamburger menu icon)
  • Responsive images (max-width: 100%, height: auto)
  • Responsive layouts (Grid, Flex, Float, Positioning adapt to screen size)
